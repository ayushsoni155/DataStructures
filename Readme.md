# Important DSA Questions (solved)

A concise index of important data-structures & algorithms problems solved during practice. Use this README to navigate, add, and review solutions.

## Structure
- /Arrays
- /Strings
- /LinkedLists
- /StacksQueues
- /Trees
- /Graphs
- /DynamicProgramming
- /Misc

## How to use
1. Open a category folder.
2. Each problem has a README or solution file named: ProblemName.md / ProblemName.(cpp|py|java).
3. Each solution includes: problem link, approach, code, time & space complexity, and notes.

## Core problem checklist (examples)
- Arrays
    - Two Sum, Subarray Sum, Merge Intervals, Dutch National Flag
- Strings
    - Longest Palindromic Substring, Anagram Check, KMP pattern match, Sliding Window problems
- Linked Lists
    - Reverse List, Detect Cycle, Merge k lists, Remove nth node from end
- Trees
    - Inorder/Preorder/Postorder, LCA, Balanced BST check, Serialize/Deserialize
- Graphs
    - BFS/DFS templates, Shortest path (Dijkstra), Topological sort, Detect cycle
- Dynamic Programming
    - Knapsack, Longest Increasing Subsequence, Coin Change, Edit Distance
